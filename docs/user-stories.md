# User Stories

## Features (Planned for 2 Weeks)

1. User registration and login.

2. View all posts on the homepage.

3. Submit a news post with a title and URL.

4. Create an Ask post to start a discussion.

5. View details of a post.

6. Upvote posts.

7. View vote count for each post.

8. Add comments to posts and reply to that comments.

9. View post author .

10. Admin can manage users.

11. Admin can manage posts and comments.

12. Edit and delete posts.

13. Search posts by keywords.

14. Trending posts section.

15. User profile page.
